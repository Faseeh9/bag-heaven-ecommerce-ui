# Bag Heaven E-Commerce UI

Bag Heaven is a clean, modern front-end web project designed for a luxury bag retail brand. This project demonstrates proficiency in building structured, multi-page websites using semantic HTML5 and custom CSS3.

## 🚀 Key Features
- **Multi-Page Navigation:** Fully linked architecture including Home, Products, Cart, About, and Contact pages.
- **E-Commerce Layout:** Includes a product catalog with 12+ items, a featured products section, and a detailed cart summary.
- **Glassmorphism UI:** Advanced CSS techniques using `backdrop-filter: blur()` and semi-transparent borders for a high-end feel.
- **Responsive Design:** Utilizes Flexbox for flexible layouts and a viewport-optimized structure for mobile and desktop.
- **Interactive Forms:** A functional contact form with styled input fields and submission buttons.

## 🛠️ Technical Stack
- **Languages:** HTML5, CSS3
- **Styling:** Custom External Stylesheet, Flexbox Layouts, Font Awesome Integration
- **Assets:** Optimized for image-heavy storefronts with dedicated product galleries.

## 📂 Project Structure
- `index.html`: The landing page featuring a hero section and featured products.
- `products.html`: Comprehensive catalog of the bag collection.
- `cart.html`: Shopping cart interface with subtotal/tax calculations.
- `contact.html`: Customer inquiry form and brand details.
- `about.html`: Brand story and social media connectivity.
- `styles.css`: Global stylesheet managing the site's unique "Dark Theme" aesthetic.
